Class: Registry

azure-iothub. Registry

Constructs a Registry object with the given IoT Hub connection string and transport object. The Registry class provides access to IoT Hub identity service.

Constructor

new Registry(connString, transport)

Parameters:
Name Type Description
connString String

A connection string which encapsulates the appropriate (read and/or write) Registry permissions.
transport Object

An object that implements the interface expected of a transport object. See the file node/common/lib/devdoc/https_requirements.docm for information on what the transport object should look like. The module:azure-iothub.Https class is one such implementation of the transport.

Parameters:
Name Type Description

The object must include a deviceId property with a valid device identifier.
done function

The function to call when the operation is complete. done will be called with three arguments: an Error object (can be null), a transport-specific response object useful for logging or debugging, and a module:azure-iothub.Device object representing the created device.
